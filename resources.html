Resources

This are almost all the resources that I have used to learn web dev related topics, the order of these is the one that I would recommend someone just starting to learn. The first item, are some curriculums that I think are pretty good and would love to finish soon (between this year and the next one).

  1. Curriculums / Complete Courses
    • Milan Petrovic - Web development learning guide (a frontend approach)
    • P1xt's Guide: Get Job ready with 1 FCC cert, 3 projects, 2 courses, and 10 books
    • FreeCodeCamp
    • The Odin Project
    • Open Source University
    • Udemy - The Web Developer Bootcamp (paid: $15)
    • Udemy - The Complete Web Developer Course 2.0 (paid $15)
      • If you are going to buy one of the Udemy's courses I would recommend The Web Developer Bootcamp by Colt Steele, he is a great teacher and the course covers Full-Stack Web Development all with JavaScript, some of the topics are; HTML, CSS, Bootstrap, JS, jQuery, node.js, express, etc

  2. How The Internet Works
    • Go to How the Internet works section in Milan Petrovic guide, read the documents (they're not so long) and watch the videos (some of them are pretty good) and you should be good to go, but if you don't want to spend much time in this section just watch the Khan Academy course on How the Internet Works that include the "Father of the Internet" Vint Cerf

  3. GNU/Linux
    • If you don't have a mac or a linux based operatyng system like Ubuntu, then read this article by The Odin Project about why you shouldn't use Windows for development (Of course you could write software in a Windows OS, but it is going to be harder) and what are the alternatives if you have this OS (install Ubuntu, use a virtual machine, etc).

  4. The Command Line
    • Do the Command Line Basics section in The Odin Project
    • (optional) the Rithm School, have a course about the Terminal and UNIX that is more advanced than The Odin Project resources tells that you should do

  5. HTML/CSS

    • Start with interactive tutorials, I would say to start with Codecademy's Make a Website and then do HTML/CSS, after that you could do Khan Academy Intro to HTML/CSS

    • Other websites that are good to learn HTML and CSS are:

      • Shay Howe's Learn to Code HTML and CSS (I recommend doing this tutorial/guide and finishing it a 100%)
      • HTML Dog's HTML Tutorials
      • Dive Into HTML5 by Mark Pilgrim (I haven't read this, but I have seen many recommendations on the internet, plus the illustrations seem really nice)
    • If you like books, this are the ones that I've finished, in the order that I would recommend reading them:
      1. Illustrated Guide to Front-End Development (paid: $15)
      2. Thinking in HTML (free)
      3. Thinking in CSS (free)
      4. How to Code in HTML5 and CSS3 (free)
      5. HTML5 For Web Designers (paid: $9)
      6. CSS3 For Web Designers (paid: $11)
      7. Head First: HTML and CSS (paid $33.99)
      8. HTML and CSS: Design and Build Websites (paid: $30)
      9. Creating a Website: The Missing Manual (paid: $5.40)
      10. CSS: The Missing Manual (paid: $4.72)

    • The only video course that I've completed (beside the two that I recommend in the Curriculum section) is Semantic HTML: How to Structure Web Pages which is free


  6. JavaScript

    • Like HTML/CSS, I think that starting with interactive tutorials is a really good starting point; first, do Codecademy's Javascript and then Khan Academy's Intro to JavaScript

    • Some really good courses that I would recommend to anyone that wanted to have a really good base in JS are:

      • Practical JavaScript by Gordon Zhu, which is free.
      • JavaScript: Understanding the Weird Parts by Tony Alicea, which cost $15 in Udemy.

    • And ultimately if you like books, you could read them in this recommended order:

      1. JS for Cats (free)
      2. Head First JavaScript Programming (paid: $42.99)
      3. Beginning Javascript (paid: $30.32)
      4. Javascript and jQuery: Interactive Front-End Web Development (paid: 23.67)
      5. Javascript the Missing Manual (paid: $42.99)
      6. JavaScript the Definitive Guide (paid: $4.99)
      7. Professional Javascript for Web Developers (paid: $29.99)
      8. You Don't Know JS: This a collection of 6 books (which all of them are free) by Kyle Simpson that cover JavaScript. I've just read the first one of these books (Up and Going) and was really good from a begginer point of view.
      9. Eloquent JavaScript: This book is also free (available online) and it is written by Marijn Haverbeke. I haven't finished it (I've just read the first couples of chapters) and people say that this book is really good, but that it can get hard really fast if you are still a beginner.
